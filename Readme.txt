本地测试请将server.py和client.py的ip地址改为127.0.0.1，端口改为13333
外网测试请将client.py的ip地址和端口分别改为47.106.11.170 和1234  （有效期至2020.8.15）

该版本为方便本地运行调试，未使用数据库

使用说明：
（1）在命令行界面运行 pip install -r requirements.txt
（2）本地测试先运行Server.py,再运行Main.py。 外网测试直接运行main.py
（3）不能用IDLE运行，建议使用pycharm或命令行运行，命令行运行需要先进入程序目录再运行
（4）最左侧为在线列表，双击列表中的项目可以发起私聊，列表中#Group#为所有在线人可见的聊天室
（5）支持回车键快捷发送消息和确认登录、注册

需要安装的第三方模块（Python3）：
PIL
