目前只在本地测试，可以通过修改server和client中的IP地址实现远程连接
使用说明：
（1）先运行Server.py,再运行Main.py
（2）最左侧为在线列表，双击列表中的项目可以发起私聊，列表中#Group#为所有在线人可见的聊天室
（3）支持回车键快捷发送消息和确认登录、注册

需要安装的第三方模块（Python3）：
PIL
tkinter
服务器（server.py）需要pymysql模块 且本地需要安装mysql
